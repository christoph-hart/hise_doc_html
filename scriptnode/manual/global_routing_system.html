The Global Routing System

HISE uses an encapsulated module system to keep things as local as possible. However there are a few use cases where you need to break the standard tree signal flow, which is why there are a few methods of communicating and / or sending signals between different modules:

  1. Parameter System: most stable option for static connections.
  2. Global Modulators: allows a sample-accurate modulation signal when applicable
  3. Send effects: implement a AUX effect section when you want to send various sources through an optional effect chain.
  4. Macro controls: allows a flexible (yet imprecise) modulation system for every parameter
